VISUAL STYLE

Use a Japandi + Bauhaus-inspired visual language:

- Clean editorial typography
- Large whitespace
- Strong 12-column grid
- Minimal geometric shapes
- No gradients
- No heavy shadows
- No decorative icons
- No unnecessary visual clutter

Typography:
- Use Inter or a clean modern sans-serif
- Large editorial headlines
- Clear hierarchy: eyebrow labels, H1, H2, H3, body, caption
- Keep body copy readable and not too dense

- Use generous vertical spacing between sections
- Keep all sections aligned to a 12-column grid
- Use calm, refined transitions visually, but generate static layout for now
